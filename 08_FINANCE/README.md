# Finance Department

## Purpose

Manage financial planning, budgeting, costing, billing, procurement, profitability, and financial reporting for every VAYRA Atelier project.

## Objectives

• Maintain financial transparency

• Improve project profitability

• Control project costs

• Standardize quotations and invoices

• Monitor cash flow

• Support business growth
