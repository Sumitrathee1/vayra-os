# Context Engine

## Purpose

The Context Engine determines what information each AI agent should load before performing a task.

Its purpose is to provide only relevant context, reducing unnecessary information while improving decision quality and response consistency.

## Objectives

• Improve AI accuracy

• Reduce irrelevant context

• Increase retrieval speed

• Standardize context loading

• Support scalable AI workflows

Every AI Agent must follow the Context Engine before answering complex requests.
